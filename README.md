# Spatialify

Extensão Firefox para adicionar efeitos de áudio espacial ao YouTube Music. Transforme sua experiência de áudio com expansão estéreo, efeito Haas e processamento local em tempo real.

## ✨ Recursos

- **Expansão Estéreo** - Amplie a imagem estéreo para uma sensação de palco mais amplo
- **Efeito Haas** - Atraso sutil entre canais para criar profundidade e espaço
- **Inversão de Fase** - Controle fino da fase do canal direito
- **Presets** - Sutil, Amplo, Ultra e presets personalizados
- **Processamento Local** - Tudo acontece no navegador, zero dados enviados

## 🚀 Instalação

1. Baixe a extensão através do link: `https://addons.mozilla.org/en-US/firefox/addon/spatialify/`

## 🎵 Como Usar

**Popup Rápido:**
- Clique no ícone Spatialify na toolbar
- Ative/desative com o toggle
- Ajuste a intensidade (0-100%)
- Escolha presets: Sutil, Amplo ou Ultra

**Painel Avançado:**
- Clique em "Avançado" no popup ou acesse → Spatialify → Preferências
- Controles detalhados: Largura Estéreo, Efeito Haas, Inversão de Fase, Compensação de Ganho
- Salve seus próprios presets

## 🔒 Privacidade

- **100% Local:** Todo processamento via Web Audio API no seu navegador
- **Zero Dados:** Nenhum dado de áudio ou configuração é enviado para servidores
- **Apenas YouTube Music:** Funciona exclusivamente em `music.youtube.com`

## 🛠️ Tecnologias

- Web Audio API
- Firefox Extension Manifest V3
- Content Scripts
- Storage API (apenas para salvar preferências localmente)

## 📋 Requisitos

- Firefox (desktop)
- YouTube Music
